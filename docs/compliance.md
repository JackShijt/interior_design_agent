# 合规文案与隐私脱敏（MVP 上线前）

> 责任归属：以下口径由产品负责人（人）拍板，agent 仅落地实现。

## 1. 用户告知（免责声明，前端展示）

```
本方案由 AI 辅助生成，户型识别与空间规划结果仅供参考。
承重结构、水电走向等涉及安全的改动，须经专业工程师与物业确认后方可施工。
因参照本方案施工导致的任何损失，平台不承担责任。
```

## 2. 隐私与数据

- 户型图仅在用户授权环境（本地/自有服务）内处理，默认不留存原始图片；
- 不收集用户姓名、手机号等敏感个人信息（如需账号体系，单独告知并取得同意）；
- 方案快照（`data/versions.json`）、审计日志（`data/audit.log`）仅存于本地，不对外同步；
- 密钥（LLM_API_KEY 等）仅通过环境变量注入，禁止写入代码或配置文件。

## 3. 安全红线（强制）

- 承重墙拆除：100% 阻断或强制人工确认，绝不自动执行；
- 管井/烟道移位：阻断；
- 承重墙开洞 >1200mm：阻断（需结构加固）。
- 所有红线操作记录审计日志，可追溯"谁、何时、确认了什么"。

## 4. 内容合规

- 构件库报价为参考价，实际以当地市场/商家为准；
- 不承诺具体施工效果，标注"AI 生成，仅供参考"。
