# 室内设计软件调研清单

> 调研时间：2026-07-14
> 调研范围：面向家装 / 工装 / 全屋定制的室内设计软件，聚焦其技术架构、核心功能与实现原理

| 类别 | 代表软件 | 技术形态 | 典型用户 |

| 云端 3D 设计平台 | 酷家乐、三维家、Homestyler（居然设计家） | Web / 云原生，SaaS | 设计师、装修公司、业主 |

## 二、室内设计软件核心技术

| 技术 | 原理 | 室内设计中的作用 | 代表软件 |
|------|------|------------------|----------|
| **云端实时渲染（云渲染）** | 服务器 GPU 集群渲染，视频流 / 图片回传终端，所见即所得 | 10 秒出高清效果图、实时打光 | 酷家乐 KooEngine、三维家 Sunvega |
| **WebGL / WebGPU 本地渲染** | 浏览器端调用本地 GPU 做实时光栅化 / 光线追踪 | 网页内轻量 3D 交互、免安装 | Homestyler、酷家乐 Web 端 |
| **户型自动识别 / 重建** | CV / OCR 识别户型图或照片，生成矢量墙线 + 3D 墙体 | 快速起稿、量房 | 知户型、酷家乐、三维家 |
| **参数化模型库 + 拖拽布局** | 家具 / 硬装构件参数化建模，搭靠吸附 / 对齐约束 | 软硬装快速布置 | 全部云平台 |
| **BIM / 数据联动** | 模型带材质 / 尺寸 / 价格属性，自动算量、出施工图 | 施工落地、报价 | 三维家、酷家乐企业版 |
| **AI 生成式设计** | 扩散模型 / 大模型，按文本或照片生成多风格方案 | 一键出图、灵感探索 | 居然设计家 AI、Homestyler AI |
| **超分渲染（AI SR）** | AI 超分辨率将低分辨率渲染放大到 4K/8K | 提升出图清晰度、省算力 | 三维家（TensorRT 超分） |
| **实时光追（RTX）** | GPU 光线追踪做实时阴影 / 反射 | 写实材质灯光预览 | D5、Enscape |

## 三、逐款软件调研

### 1. 酷家乐（群核科技）
- **技术架构**：云原生 SaaS，自研 **KooEngine** 渲染引擎；前端 WebGL，后端万级 GPU 集群；分布式户型几何计算服务（kam，CPU 密集，动态扩缩容）。
- **核心功能**：
  - 3 步上手、10 秒出超清 3D 效果图；
  - 户型绘制 / 导入、硬装（瓷砖 / 吊顶 / 橱柜）参数化；
  - **云端实时打光渲染**（行业首发），毫秒级实时预览，效率提升约 10 倍；
  - 720° 全景、VR、8K 输出、施工图 / 算量。
- **实现要点**：渲染在云端 GPU 完成，终端只接收流 / 图；海量素材库（数亿模型）云端管理；户型业务做独立计算服务解耦。
- **适用**：装修公司、全屋定制、软装、房产展示。

### 2. 三维家
- **技术架构**：云端架构，PC 端网页即开即用；基于 **NVIDIA GPU + TensorRT** 推理加速；自研 **Sunvega 3D 垂类大模型**。
- **核心功能**：拖拽式交互、自动吸附；硬装 / 软装 / 全屋定制；AI 超分渲染（整体时间节省 50%+）；移动端量房 App；BIM 级数据联动。
- **实现要点**：3D 垂类大模型驱动设计与渲染；AI 超分把低分辨率渲染放大到高清，显著降低算力成本；设计历史 / 参数化构件。
- **适用**：定制家居企业、工厂拆单、终端门店设计。

### 3. Homestyler / 居然设计家
- **技术架构**：云端 3D 平台（原 HD 中美团队，现居然智家），WebGL 渲染 + 云端。
- **核心功能**：免费在线绘制平面图、2D/3D 切换、VR；**AI 装饰**（按风格自动布局建议）；上传房间照片，AI 一键生成多达 29 种风格效果图；海量真实品牌家具模型、AR 预览。
- **实现要点**：模板 / 上传双入口；AI 风格生成基于图像大模型；全球设计师社区 + 多语言 + 电商联动（设计即购买）。
- **适用**：海外用户、业主自助、软装灵感。

### 4. 知户型
- **技术架构**：移动 / Web 轻量端，App 为主；CV 辅助量房。
- **核心功能**：手机量房、自动生成户型图与 3D；硬装 / 软装布置；导出 CAD / 报价；轻量快速出图。
- **实现要点**：以量房入口切入，降低现场测量门槛；轻量渲染，适合外勤。
- **适用**：设计师、工长、门店快速量房出方案。

### 5. 酷大师（酷家乐旗下）
- **技术架构**：在线 3D 建模，云端建模 + 高清渲染。
- **核心功能**：自由建模、实时渲染，定位"让建模更高效简单"；面向定制化建模需求。
- **实现要点**：基于酷家乐云渲染能力，扩展自由建模（不仅模板化）。
- **适用**：需要自定义建模的设计师、定制企业。

### 6. SketchUp + D5 Render / Enscape
- **技术架构**：桌面端（SketchUp Ruby 建模）+ 实时渲染插件（D5 基于 UE/实时光追，Enscape 基于实时光栅 + RTX）。
- **核心功能**：推拉直觉建模做硬装结构；D5 / Enscape 实时光追材质灯光预览；联动 PS 后期。
- **实现要点**：专业建模 + 第三方实时渲染器组合，色彩 / 灯光可控性强，适合精品方案。
- **适用**：专业室内设计师、效果图公司。

### 7. 3ds Max + Corona / V-Ray
- **技术架构**：桌面端（3ds Max 修改器栈）+ 离线 / 半实时渲染器（Corona 路径追踪、V-Ray）。
- **核心功能**：精细建模、照片级效果图、复杂的材质灯光；施工图配合。
- **实现要点**：离线高质量渲染，出图精细但慢；适合最终交付级效果图。
- **适用**：高端效果图、工装、酒店。

### 8. 美间
- **技术架构**：Web 端软装设计工具，模板化 + 素材库。
- **核心功能**：软装方案拼搭、清单报价、PPT 级方案输出；主打软装陈列与采购。
- **实现要点**：以软装素材 + 模板降低出方案门槛，衔接电商采购。
- **适用**：软装设计师、家居买手。

## 四、技术实现共性总结

1. **云原生成为主流**：酷家乐 / 三维家 / Homestyler 均把渲染、计算、素材放到云端，终端轻量化（网页 / App），靠 GPU 集群提供"秒级出图"。
2. **自研渲染引擎 + GPU 加速**：酷家乐 KooEngine、三维家 Sunvega 以 GPU 集群 + 实时打光实现所见即所得。
3. **AI 深度融入**：户型识别（CV）、AI 风格生成（扩散模型）、超分渲染（TensorRT）、3D 垂类大模型，全面降本增效。
4. **参数化模型库 + 数据联动**：构件带尺寸 / 材质 / 价格属性，一键算量、出施工图、报价的 BIM 化趋势明显。
5. **工作流分层**：
   - 快速揽客 / 门店：酷家乐、三维家（云、快、模板）；
   - 专业精品：SketchUp + D5、3ds Max + Corona（可控、精细）；
   - 业主自助 / 灵感：Homestyler AI、居然设计家 AI。
